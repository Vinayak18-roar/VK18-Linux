## Day1 practiced basics of linux:

- pwd     : Print working directory.
- cd      : Chnage directory.
- cd ..   : It will bring to parent directory.
- cd .    : To be in current directory.
- cd /    : To go to root directory.
- cd ~    : To go to Home directory
- ls      : To check store lists of any folders.
- ls -l   : lists in long formatting detailed list (raw size bytes).
- ls -lh  : lists in long formatting detailed list (Human readable size).
- ls -a   : To check hidden files.
